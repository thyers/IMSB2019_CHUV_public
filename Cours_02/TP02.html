Rponse 2.1

La tartine n'ayant que deux faces, le rsultat s'apparente un tirage sur une loi binomiale de paramtre \( \theta \) inconnu. Quelle est la distribution posterieure du paramtre \( \theta \) associe ces donnes sachant que l'analyste n'avait aucun a priori.

  • Il faut commencer par construire un tibble qui contient les domnnes :
nbTrial <- length(data$trial)
nbSucces <- sum(data$value)
grid_size <- 1e3

df <- tibble( 
  p_grid     = seq(from = 0., to = 1., length.out = grid_size),
  prior      = rep(1, grid_size)) %>%
  mutate(likelihood = dbinom(nbSucces, size = nbTrial, prob = p_grid)) %>%
  mutate(posterior  = likelihood * prior) %>%
  mutate(posterior  = posterior / sum(posterior))

# lower right panel
df %>% 
  ggplot(aes(x = p_grid)) +
  geom_ribbon(data = df %>% filter(p_grid > hdi(samples$p_grid, .width = .95)[1] & 
                                    p_grid < hdi(samples$p_grid, .width = .95)[2]),
              aes(ymin = 0, ymax = posterior),
              fill = "grey75") +
  geom_line(aes(y = posterior)) +
